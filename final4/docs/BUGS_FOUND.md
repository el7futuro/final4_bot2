## Ошибка 1: Структура хода

**Текущая реализация:**
- ~~Каждый игрок ходит по очереди (сначала один, потом другой)~~
- ~~Каждый бросает кубик отдельно~~

**Правильно:**
- ОБА игрока делают ставки одновременно в рамках одного хода
- ОДИН бросок кубика определяет результат для ОБОИХ

**Статус:** ✅ ИСПРАВЛЕНО
- TurnState обновлён для хранения ставок обоих игроков
- GameEngine.roll_dice() теперь принимает только match (без manager_id)
- Добавлен метод confirm_bets() для завершения ставок
- Добавлен метод can_roll_dice() для проверки готовности обоих

## Ошибка 2: Количество ставок за ход

**Текущая реализация:**

- ~~Вторая ставка опциональна~~

**Правильно:**
- Ход 1 (вратарь): РОВНО 1 ставка

**Статус:** ✅ ИСПРАВЛЕНО
- TurnState.get_required_bets_count() используется
- confirm_bets() проверяет количество ставок
- place_bet() проверяет, что ставки на одного игрока

## Ошибка 6: Две ставки одного типа ЗАПРЕЩЕНЫ

**Текущая реализация:**
- ~~Можно делать 2 ставки любого типа на одного игрока~~

**Правильно:**
- НЕЛЬЗЯ 2 ставки одного типа (например, Чёт/Нечёт + Чёт/Нечёт)
- МОЖНО комбинации разных типов:
  - Чёт/Нечёт + Больше/Меньше ✅
  - Чёт/Нечёт + Точное число ✅
  - Больше/Меньше + Точное число ✅

**Статус:** ✅ ИСПРАВЛЕНО
- BetTracker.validate_bet() проверяет типы при второй ставке
- Добавлен _get_player_bet_type_this_turn()

## Ошибка 7: Не проверяются лимиты перед показом вариантов

**Текущая реализация:**
- ~~Показываю все типы ставок, потом проверяю лимиты~~

**Правильно:**

**Статус:** ✅ ИСПРАВЛЕНО

- Лимиты проверяются до формирования списка

## Ошибка 8: КРИТИЧЕСКАЯ — Игрок доступен только если есть 2 разных типа ставок

**Текущая реализация:**
- ~~Показываю всех игроков, у которых is_available=True и не использованы~~

**Правильно:**
- Игрок доступен ТОЛЬКО если у него есть минимум **2 РАЗНЫХ типа ставок**
- Если из-за лимитов остался только 1 тип — игрок **НЕДОСТУПЕН**
- Проверка должна быть ДО показа списка игроков

**Статус:** ✅ ИСПРАВЛЕНО
- get_available_players() теперь проверяет количество типов
- can_player_bet() возвращает ошибку при < 2 типах
- Тест test_player_with_one_type_not_available проходит

**Оптимизация от пользователя:**
