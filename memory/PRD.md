## РЕАЛИЗОВАНО

### Состав команды
- 1 GK + 5 DF + 6 MF + 4 FW = **16 игроков**

### Основное время (11 ходов)
- Ход 1: Вратарь (1 ставка на чёт/нечёт)
- Ходы 2-11: Полевые (2 ставки РАЗНЫХ типов)
- Лимиты: 1 DF гол, 3 MF гола, 4 FW гола, 6 чёт/нечёт

### Дополнительное время (5 ходов)
- Все 5 ходов — ПОЛЕВЫЕ игроки (вратарь уже использован)
- **ОБЯЗАТЕЛЬНО**: 1 ставка на гол + 1 позиционная (чёт/нечёт или больше/меньше)
- Форварды: гол + больше/меньше (не чёт/нечёт)
- Лимиты НЕ действуют — каждый может ставить на гол
- Статистика ET считается ОТДЕЛЬНО от Main Time!

### Колода карточек (50 шт.)
| Карточка | Кол-во | Цель | Эффект |
|----------|--------|------|--------|
| Хет-трик | 1 | Свой | +3 гола |
| Дубль | 1 | Свой | +2 гола |
| Гол | 2 | Свой | +1 гол |
| Автогол | 2 | Соперник | +1 гол сопернику |
| ВАР | 2 | Особая | Отменяет карточку |
| Офсайд | 2 | Соперник | Отменяет гол |
| Пенальти | 3 | Особая | Интерактивный розыгрыш |
| Удаление | 2 | Соперник | Все действия = 0 |
| Предупреждение | 3 | Соперник | -1 действие |
| Фол | 8 | Свой | -1 отбитие |
| Потеря | 8 | Свой | -1 передача |
| Перехват | 8 | Свой | +1 передача |
| Отбор | 8 | Свой | +1 отбитие |

### Расчёт счёта
**Формула:**
1. Передачи съедают отбития соперника (1:1)
2. Каждый гол "съедает" до 2 отбитий, но сам НЕ проходит пока есть отбития
3. Гол засчитывается ТОЛЬКО когда отбитий = 0

`goals_scored = max(0, own_goals - ceil(remaining_saves / 2))`

### Серия пенальти (автоматическая)
- Порядок: Extra Time (обратный) → Main Time (обратный)
- Есть передача → ГОЛ
- Нет передачи → промах

### Интерактивная карточка Пенальти
- Выбор High/Low → Кнопка "Бросить кубик" → Показ результата
- Работает и для PvBot и для PvP (соперника уведомляют)

### PvP Matchmaking
- Тип: `MatchType.RANDOM`
- Синхронизация FSM через in-memory storage
- Только manager1 бросает кубик
- Manager2 уведомляется через send_message
